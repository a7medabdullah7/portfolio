A modern, dark-themed portfolio built with **Next.js 14**, **React**, **TailwindCSS**, and **Framer Motion**.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Add Your Assets

1. **CV (Download button)**  
   Place your PDF resume at `public/cv.pdf`.

2. **SmartANPR project image**  
   Place your SmartANPR screenshot/image at `public/images/smartanpr.jpg` (or `.png`).  
   If the file is missing, the site shows a gradient placeholder and still works.

## Build

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TailwindCSS
- Framer Motion
- TypeScript
